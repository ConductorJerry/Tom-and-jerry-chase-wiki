# 米特操作技巧

以下内容中胡椒粉简称为粉，爆炸简称为烟。

1. 烟的第一段伤害存在延迟，意味着鼠能够通过立即回血或长时间免虚弱和无敌免死
2. 粉可以被老鼠接住，无法形成烟；受部分击退且掉落鼠，则鼠原地下来，粉被撞飞，无法形成烟造成虚弱；跳在空中时，若受到掉落鼠的攻击，粉下落比鼠慢，形成烟的时间可以让鼠逃跑
3. 烟中的虚弱鼠若被抓起，则抓起后鼠在手中虚弱，虚弱时间内无论以任何办法下手或下火箭，仍为虚弱状态，若在虚弱时与抓起的间隔中铁血CD读秒完成，则在手上铁血，绑上火箭仍然铁血，铁血结束不虚弱
4. 二级粉放下鼠，老鼠2秒内(此时间为眩晕时间)不会禁技能，但是2秒后禁技能，时间约等于猛攻，且附带高额减速
5. 手上有一个粉，且2级粉CD好，则可以原地放粉，将手上的鼠放下来，转头攻击其他离得近的鼠，容易打出双倒
6. 粉放下鼠，由于烟第一段伤害有延迟，可以打出爪刀，用于刷经验和三被动回血，挂二被动的禁疗
7. 有3被和2级粉，此时剩两人，抓住一只可以一直刷虚弱，稳赢
8. 二级粉放下鼠，若鼠有护盾，则不会眩晕(即使脚下有粉也不能)，可以秒按治疗回血
9. 开局向洞口丢烟放夹，破机器鼠
10. 手中鼠如果被天使上复活，用粉放下来，重新抓起可以刷掉
